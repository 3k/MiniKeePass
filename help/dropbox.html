Adding files from Dropbox

KeePass files located on Dropbox can be added to MiniKeePass using the Dropbox iPhone app.

Steps:

  • Open the Dropbox iPhone app
  • Navigate to your KeePass file
  • Click on the KeePass File
  • Click the Open With button
  • Select KeePass

Exporting to Dropbox

Files updated in MiniKeePass will not automatically sync back to Dropbox. For Dropbox to reflect changes made in MiniKeePass the file must be exported.

Steps:

  • Open the file to be exported
  • Tap on the Action arrow in the toolbar
  • Select "Dropbox" from the list that appears
